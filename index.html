Introduction

TypeScript is a helpful programming language in web development. TypeScript has all of the same functionality as JavaScript with some extra features. The biggest difference is that Typescript is strongly typed and allows for static typing while javascript does not.

Fundamentals

Variables

Variables can be assigned by using the let keyword as shown below. You can also create a constant by using the const keyword. A constant's value cannot be changed. Typescript will infer the type of a vairable for you.

Conditionals

Typecript supports if - else if - else trees for evaluating conditional values. Typescript has an equality comparion operater in '==' and an identity comparison operator in '==='. The difference between Javascript and Tyepscript here is that because strong typing is enforced both operators function similarly.

Loops

A while loop executes code after checking if a condition is true. A do - while loop executes code before checking if a condition is true. A for loop can executes code using an iterator or for a certain number of iterations.

Functions

Functions are defined using the function keyword followed by the function name and parenthesis that hold parameters and their types. If the function has a return value, the type of the return value is specified by the ':' symbol after the parameters. If the type of a parameter or return value is not specified it automatically gets the type 'any', but this is not good practice.
